    {{% timeline-item date="2016-11-15" month="November" title="SeedStack 16.11 \"Kiwi\"" %}} SeedStack 16.11 "Kiwi" will include the following features: * Completely new configuration system, * New version of W20, compatible with AngularJS 2, * OAuth support, * Cloud configuration add-on, * Circuit Breaker add-on, * Service discovery add-on. {{% /timeline-item %}} {{% timeline-item date="2016-07-15" month="July" title="SeedStack 16.7 \"Jasmine\"" %}} SeedStack 16.7 "Jasmine" will be a maintenance release. {{% /timeline-item %}} {{% timeline-item date="2016-05-19" month="May" title="Zenika NightClazz DDD" %}} Hands-on of SeedStack and its Business Framework at the [NightClazz DDD meet'up](http://www.meetup.com/fr-FR/NightClazz/events/229957934/) at Zenika, Paris. {{% /timeline-item %}} {{% timeline-item date="2016-04-28" month="April" title="SeedStack 16.4 \"Iris\"" %}} SeedStack 16.4 "Iris" includes the following features: * Fully Java 8 compatible, * Better Web integration, * Asynchronous REST resources and Servlets, * [MQTT](http://mqtt.org/) messaging add-on, * New Material design theme, * High-volume zoomable line chart, based on [Dygraphs](http://dygraphs.com/). {{% /timeline-item %}} {{% timeline-item date="2016-03-03" month="March" title="SeedStack 15.11.3" %}} This intermediate version streamlines the project templates. Particularly, the Web project template has been refactored as a single-module project. {{% /timeline-item %}} {{% timeline-item date="2016-02-09" month="February" title="SeedStack 15.11.2" link="/posts/15-11-2-release-notes" %}} This intermediate version includes fixed MongoDB, Solr, Neo4J and Redis modules which were not released correctly. It also include a newer version of w20-bridge. {{% /timeline-item %}} {{% timeline-item date="2016-01-21" month="January" title="SeedStack 15.11.1" link="/posts/15-11-1-release-notes" %}} This intermediate version includes the following features and improvements: * Tightened security, * JAX-RS 2 support (through Jersey 2), * [Swagger](http://swagger.io/) add-on, * Improved i18n performance, * Upgrade to the latest Angular 1.x version. {{% /timeline-item %}} {{% timeline-item date="2015-11-26" month="November" title="SeedStack 15.11 \"Hibiscus\"" link="/posts/15-11-release-notes" %}} This version comes with a greatly improved add-on architecture and several new features: * [MongoDB](https://www.mongodb.org/) persistence, * [Redis](http://redis.io/) persistence, * [Neo4J](http://neo4j.com/) persistence, * [Solr](http://lucene.apache.org/solr/) persistence, * Business framework improvements, * Cryptography support, * Injectable domain registry. {{% /timeline-item %}} {{% timeline-item date="2015-11-19" month="November" title="Paris Open-Source Summit" image="img/logo-poss.png" %}} We will be presenting SeedStack at the Paris Open-Source Summit conference. Check the [conference agenda](http://opensourcesummit.paris/preinscription-conferences.html?orderby=time&step=1) for details. {{% /timeline-item %}} {{% timeline-item date="2015-07-28" month="July" title="SeedStack 15.7 \"Gingko\"" %}} First public stable release of SeedStack. {{% /timeline-item %}} {{% timeline-item date="2015-02-01" month="February" title="Creation of SeedStack OSS" %}} Creation of the GitHub organization. SeedStack is an Open-Source enterprise-class development stack derived from the former PSA Peugeot Citroën internal development stack. This internal version was used for two years by more than 20 projects. {{% /timeline-item %}} {{% timeline-item date="2013-10-14" month="October" title="First internal release of Seed" %}} After 10 months of development with multiple beta releases, the first stable version of Seed, the Java framework, is released internally, at PSA Peugeot Citroën. {{% /timeline-item %}} {{% timeline-item date="2012-09-28" month="September" title="First internal release of W20" %}} The first stable version of W20 is released. Since there was no Seed framework at the time, it was used with another, already existing backend framework. {{% /timeline-item %}}
